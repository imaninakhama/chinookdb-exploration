ORDERING
This is the sorting of data after grouping and filtering
In ordering ascending version is the default
Other ways in which ordering is done 

*descending (DESC)*
From top to bottom

*multiple columns*
seperated by commas

*aggregate*
where many rows collapses them into a single summary value

*position*
sorting by column index

*combining where&orderby*
where narrow down the data 
and order by now sorts that smaller data
