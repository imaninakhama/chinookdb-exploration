FILTERING
 Showing the rows that really matter and hiding the rest
 We use the query WHERE (sometimes followed by conditions)

 The following are examples of conditions
 *AND*
 both conditions must be true
 

 *OR*
 at least one condition must be true

 *between*
 value falls within a specific range

 *IN*
 Matches any value in a list

*LIKE*
Looks alike on text

*IS NULL*
Is used to check fpr missing data

*NOT*
flips a condition

*AND & OR*
Uses a parentheses
